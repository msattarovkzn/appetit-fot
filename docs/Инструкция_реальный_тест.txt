  АППЕТИТ ФОТ — ИНСТРУКЦИЯ ДЛЯ РЕАЛЬНОГО ТЕСТА
  Режим: LIVE_TEST_MODE (реальное время, staging окружение)
═══════════════════════════════════════════════════════════════

Сайт: https://appetit-fot.vercel.app

ВАЖНО: В режиме реального теста дата и время фиксируются
автоматически. Менять их вручную нельзя.

──────────────────────────────────────────────────────────────
  РАЗДЕЛ 1 — ОТКРЫТИЕ СМЕНЫ СОТРУДНИКОМ
──────────────────────────────────────────────────────────────

Адрес: https://appetit-fot.vercel.app/shift

1. Зайти на сайт по ссылке выше.
2. Выбрать свой филиал.
3. Ввести PIN-код на цифровой клавиатуре.
4. Нажать «Открыть смену».
5. Система автоматически запишет текущее время открытия.
6. На экране появится: «Иванов Иван, твоя смена открыта.
   Время открытия: HH:MM»
7. Нажать «Готово».

Правило: Нельзя открыть вторую смену в тот же день самостоятельно.
Если смена уже закрывалась — дополнительную смену открывает кассир.

──────────────────────────────────────────────────────────────
  РАЗДЕЛ 2 — ЗАКРЫТИЕ СМЕНЫ СОТРУДНИКОМ
──────────────────────────────────────────────────────────────

Адрес: https://appetit-fot.vercel.app/shift

1. Зайти на сайт по ссылке выше.
2. Выбрать свой филиал.
3. Ввести PIN-код.
4. Система покажет: «Смена открыта с HH:MM. Отработано: X ч.»
5. Нажать «Закрыть смену».
6. Система автоматически запишет время закрытия и посчитает
   фактические часы и минуты.
7. На экране появится: «Иванов Иван, твоя смена закрыта.
   Отработано: X ч Y мин»
8. Нажать «Готово».

──────────────────────────────────────────────────────────────
  РАЗДЕЛ 3 — ОТКРЫТИЕ СМЕНЫ КАССИРОМ
──────────────────────────────────────────────────────────────

Адрес: https://appetit-fot.vercel.app/cashier

1. Зайти на сайт.
2. Выбрать филиал.
3. Ввести PIN кассира.
4. Система покажет имя кассира и статус его смены.
5. Если смена не открыта — нажать «Открыть смену».
6. Смена откроется по текущему времени.

──────────────────────────────────────────────────────────────
  РАЗДЕЛ 4 — ЗАКРЫТИЕ СМЕНЫ КАССИРОМ И ВНЕСЕНИЕ ВЫРУЧКИ
──────────────────────────────────────────────────────────────

Адрес: https://appetit-fot.vercel.app/cashier

1. Войти с PIN кассира.
2. Нажать «Закрыть смену и день».
3. Заполнить форму:
   - Выручка (₽) — сумма выручки за СВОЮ смену
   - Количество заказов — за свою смену
   - Выносы — за свою смену (только аналитика, не влияет на расчёт)
   - Комментарий — необязательно
4. Нажать «Закрыть день».
5. Система:
   - Запишет время закрытия смены кассира
   - Создаст кассовый отчёт за день
   - Посчитает ФОТ по всем закрытым сменам
   - Посчитает бонус кассира (7₽/заказ в Пн-Чт+Вс, 5₽ в Пт-Сб)
   - Отправит уведомление в Telegram

ВАЖНО: Если два кассира работают в один день — каждый вводит
свою выручку и заказы. Данные суммируются автоматически.

──────────────────────────────────────────────────────────────
  РАЗДЕЛ 5 — ОТКРЫТИЕ ДОПОЛНИТЕЛЬНОЙ СМЕНЫ КАССИРОМ
──────────────────────────────────────────────────────────────

Ситуация: Сотрудник уже отработал и закрыл смену, но нужно
вышел ещё раз (например, подменить коллегу).

1. Кассир заходит на /cashier со своим PIN.
2. После входа внизу нажать «➕ Открыть доп. смену сотруднику».
3. Выбрать сотрудника из списка.
4. Указать время начала (или оставить пустым — запишется сейчас).
5. Указать причину (необязательно).
6. Нажать «Открыть».
7. Смена открывается с флагом «Дополнительная».
8. Когда сотрудник закончил — нажать «Закрыть смену».

──────────────────────────────────────────────────────────────
  РАЗДЕЛ 6 — ЧТО ДЕЛАТЬ, ЕСЛИ СОТРУДНИК ЗАБЫЛ ЗАКРЫТЬ СМЕНУ
──────────────────────────────────────────────────────────────

Вариант А (через кассира):
1. Кассир заходит на /cashier.
2. Нажимает «➕ Открыть доп. смену» → видит список сотрудников.
   (там можно найти незакрытую смену через бухгалтера)

Вариант Б (через бухгалтера) — РЕКОМЕНДУЕТСЯ:
1. Бухгалтер заходит на /admin → «✅ Проверка дней».
2. Выбирает вчерашнюю дату → выбирает филиал.
3. В таблице смен находит незакрытую смену (красная строка).
4. Нажимает «🔒 Закрыть» — вводит фактические часы + комментарий.
5. Система пересчитает ФОТ автоматически.

──────────────────────────────────────────────────────────────
  РАЗДЕЛ 7 — КАК БУХГАЛТЕР ПРОВЕРЯЕТ ФОТ
──────────────────────────────────────────────────────────────

Адрес: https://appetit-fot.vercel.app/admin

Логин: accountant1 / accountant123

Ежедневная проверка (утренний обход):
1. Нажать «✅ Проверка дней».
2. Выбрать вчерашнюю дату.
3. Красные филиалы — требуют внимания.
   Жёлтые — ожидают проверки.
   Зелёные — уже проверены.
4. Кликнуть на каждый филиал → правая панель покажет:
   - Выручку, заказы, средний чек
   - ФОТ % (норма: общий <27.5%, кухня <14.5%)
   - Таблицу смен всех сотрудников
5. Если всё в норме — нажать «✅ Проверка завершена».

──────────────────────────────────────────────────────────────
  РАЗДЕЛ 8 — ПРОСМОТР РЕАЛЬНЫХ ЧАСОВ СОТРУДНИКА
──────────────────────────────────────────────────────────────

Способ 1 (бухгалтер через /admin):
- /admin → таб «🕐 Смены» → выбрать филиал + период → Загрузить
- Видно: начало, конец, фактические часы, начисление

Способ 2 (бухгалтер через проверку дня):
- /admin/review → выбрать дату и филиал
- В таблице смен: «Факт время» + «Часов»

Способ 3 (сам сотрудник через личный кабинет):
- https://appetit-fot.vercel.app/employee
- Вход: логин (задаётся бухгалтером) + PIN
- Таб «🕐 Смены» → видит свои смены за месяц

──────────────────────────────────────────────────────────────
  РАЗДЕЛ 9 — РАСЧЁТ ЧАСОВ (ФОРМУЛА)
──────────────────────────────────────────────────────────────

total_minutes = closed_at − opened_at (разница в минутах)
approved_hours = total_minutes / 60

Пример:
  Открыта:  10:15
  Закрыта:  18:45
  Итого:    510 минут = 8.5 часов

Начисление (почасовой):
  base_pay = (ставка / 60) × total_minutes

Начисление (фиксированный день):
  base_pay = fixed_daily_rate (не зависит от часов)

──────────────────────────────────────────────────────────────
  РАЗДЕЛ 11 — НАСТРОЙКИ ДЛЯ ВКЛЮЧЕНИЯ LIVE TEST MODE
──────────────────────────────────────────────────────────────

На Vercel установить ENV-переменную:
  NEXT_PUBLIC_LIVE_TEST_MODE = true

После этого:
- На /shift и /cashier появится бейдж «🟢 Реальный тест»
- Тестовые UI (выбор даты, ввод часов) будут скрыты
- Все смены создаются по текущему времени сервера

NEXT_PUBLIC_TEST_MODE можно оставить = true (для /test страницы)
ENVIRONMENT=staging оставить (тестовые эндпоинты доступны)
