# Рабочие правила проекта ФОТ

Основная рабочая папка:

D:\MVP-Appetit-FOT

Только эта папка используется для:

* разработки ФОТ;
* смен сотрудников;
* расчёта зарплаты;
* деплоя;
* GitHub;
* новых функций;
* исправления ошибок.

Перед началом работы:

git status
git pull origin main

После завершения работы:

git add ...
git commit -m "..."
git push origin main

Стабильная контрольная точка:

v0.1-stable

Восстановление:

git checkout v0.1-stable

Запрещено:

* работать с ФОТ из старых копий;
* создавать новые копии проекта без необходимости;
* вносить изменения вне D:\MVP-Appetit-FOT.

Папка D:\MVP Аппетит не является рабочей папкой проекта ФОТ.

Папки backend_OLD_FOT и frontend_OLD_FOT являются архивными и используются только для справки.
